## 기능 구현

### 랜덤 세자리수 생성 기능

- 1-9 중 서로 다른 세개의 수로 이루어진 랜덤한 수

### 사용자 입력값 받는 기능

### 사용자 입력값 유효성 검사 기능

- (e) 1-9 중 서로 다른 수
- (e) 세자리수

### 점수 결과 계산 기능

- 스트라이크: 같은 수, 같은 위치
- 볼: 같은 수, 다른 위치
- 낫싱: 같은 수가 하나도 없는 경우

### 점수 결과 출력 기능

- 틀릴 경우 다시 사용자 입력값 받음
- 맞을 경우 게임 종료

### 종료 조건 확인 기능

- 1 입력 시 재시작
- 2 입력 시 종료
